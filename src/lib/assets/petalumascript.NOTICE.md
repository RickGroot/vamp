# Petaluma Script

`petalumascript.woff2` is the **Petaluma Script** font, used to render chord
symbols in the notation view in a jazz lead-sheet style.

- Designer: Anthony Hughes / Steinberg Media Technologies GmbH
- License: **SIL Open Font License, Version 1.1** (OFL) — free to use, bundle and
  redistribute with software.
- Source: bundled with [VexFlow](https://github.com/0xfe/vexflow) (extracted from
  its embedded font data via `scripts`/`node` — see commit history); upstream:
  <https://github.com/steinbergmedia/petaluma>

The full OFL text is available at <https://openfontlicense.org/>. This font is a
bundled asset, never sold on its own, in keeping with the OFL.
